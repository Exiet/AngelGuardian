## 一、音频总览

### 1.1 音频分类

| 分类 | 数量估算 | 格式 | 备注 |
|------|---------|------|------|
| UI交互音效 | ~30条 | WAV/OGG | 短音效（<1秒） |
| 卡牌释放音效 | 70条 | WAV/OGG | 部分共享基础音 + 变调 |
| 武器命中音效 | 18种×3（轻/中/重） | WAV/OGG | 含暴击变体 |
| 婴儿情感音效 | 5状态×3（进入/持续/退出） | WAV/OGG | 持续音为循环 |
| 敌人音效 | 10种×2（攻击/死亡） | WAV/OGG | Boss有专属音效 |
| 地形活化音效 | 5阶段×2（开始/结束） | WAV/OGG | 灾变有专属音效 |
| 环境音效 | 3层（基础/紧张/寂静） | WAV/OGG | 循环，可叠加 |
| 背景音乐 | 5首（主菜单/战斗/BOSS/失败/胜利） | OGG | 循环，有Intensity层叠 |

### 1.2 音频技术规范

| 项目 | 规格 |
|------|------|
| 采样率 | 44100Hz |
| 位深度 | 16bit |
| 声道 | 立体声（音效）/ 立体声（音乐） |
| 格式 | WAV（原始）/ OGG（压缩，q=6） |
| 文件大小上限 | 音效<500KB / 音乐<5MB |
| 同时播放上限 | 手机端16轨 / PC端32轨 |

## 二、婴儿情感状态音效 ⭐ 核心

婴儿是游戏的情感核心，**情感状态音效是最优先实现的音频**。

### 2.1 好奇状态（Curious）

| 音效类型 | 描述 | 时长 | 优先级 |
|---------|------|------|--------|
| 进入音效 | 婴儿"咦？"的好奇轻声 + 钟琴高音 | 0.8秒 | 高 |
| 持续环境音 | 轻微的"咯咯"笑声 + 风铃声 | 循环，每次15秒 | 中 |
| 退出音效 | 短暂的"嗯？"声 | 0.5秒 | 中 |
| 触发互动音效 | 婴儿碰到有趣物体时的"哇！"声 | 0.6秒 | 高 |

**设计意图**：让玩家听到"好奇"时，知道婴儿在探索，暂时不需要过度保护。

### 2.2 恐惧状态（Fearful）⚠️ 关键

| 音效类型 | 描述 | 时长 | 优先级 |
|---------|------|------|--------|
| 进入音效 | 婴儿"呜哇！"哭声 + 低音鼓 | 1.0秒 | **最高** |
| 持续环境音 | 断续哭泣 + 心跳声加速 | 循环，每次10秒 | **最高** |
| 退出音效 | 啜泣声逐渐平息 | 1.2秒 | 高 |
| 往天使跑音效 | 婴儿脚步声加快 + 喘息声 | 持续 | 高 |

**设计意图**：恐惧音效是**最重要的预警信号**——玩家可能在专注攻击，听到这个音效就要立刻注意婴儿位置！

### 2.3 愤怒状态（Angry）

| 音效类型 | 描述 | 时长 | 优先级 |
|---------|------|------|--------|
| 进入音效 | 婴儿"呀！"怒吼 + 钹片敲击 | 0.7秒 | 高 |
| 持续环境音 | 低沉的"呼噜"声（像小猫生气） | 循环，每次8秒 | 中 |
| 退出音效 | 怒气消散音效（"呼～"） | 0.8秒 | 中 |
| 愤怒攻击音效 | 婴儿攻击时的"哈！"声 | 0.4秒 | 高 |

**设计意图**：愤怒是"帮助玩家输出"的状态，音效要让玩家感到"爽"而不是"紧张"。

### 2.4 疲惫状态（Tired）

| 音效类型 | 描述 | 时长 | 优先级 |
|---------|------|------|--------|
| 进入音效 | 婴儿"嗯……"打哈欠声 | 1.2秒 | 中 |
| 持续环境音 | 轻微的呼吸声 + 摇篮曲旋律碎片 | 循环，每次20秒 | 低 |
| 退出音效 | 精神恢复的"叮"声 | 0.5秒 | 中 |

**设计意图**：疲惫状态音效应该让玩家感到"需要保护"，但不要太焦虑。

### 2.5 觉醒状态（Awakening）🌟 高光时刻

| 音效类型 | 描述 | 时长 | 优先级 |
|---------|------|------|--------|
| 进入音效 | 婴儿"啊——！"觉醒声 + 圣光共鸣音效 + 管风琴高音 | 2.5秒 | **最高** |
| 持续环境音 | 天使合唱 + 光之竖琴 | 循环，每次30秒 | 高 |
| 退出音效 | 光效消散音 + 婴儿满足的"嗯～" | 1.5秒 | 高 |
| 每次击杀音效 | 每次婴儿击杀敌人时的"哈利路亚"合唱片段 | 0.8秒 | **最高** |

**设计意图**：觉醒是游戏最爽的时刻，音效要**压倒性地华丽**，让玩家感到"这就是战神婴儿！"

## 三、武器连携音效

### 3.1 连携触发通用音效

所有连携触发时，先播放一个**连携启动音**（0.3秒）：
- 声音描述：水晶碰撞声 + 空间扭曲音效

### 3.2 各连携组合音效

| 连携组合 | 触发音效描述 | 持续音效 | 特殊表现 |
|---------|-------------|---------|---------|
| 冰霜+火焰（蒸汽爆炸）| 蒸汽"嘶——"声 + 爆炸低频 | 蒸汽持续2秒 | 音效有立体声平移（从左到右） |
| 圣光+荆棘（光棘螺旋）| 神圣合唱 + 荆棘生长摩擦声 | 螺旋音效持续1秒 | 音高逐渐升高 |
| 圣光+手枪（弹幕同步）| 冲锋号角 + 机枪上膛声 | 攻速提升有"呼呼"风声 | 音效速度与攻速成正比 |
| 圣光+毒（毒雨）| 圣歌 + 毒液滴落声 | 雨声持续3秒 | 雨声有空间混响 |
| 冰霜+雷电（冰雷风暴）| 冰裂声 + 雷鸣 | 风暴持续2秒 | 雷鸣有低频震动感 |

### 3.3 连携冷却音效

连携进入冷却时，播放一个**冷却完成音**（0.5秒）：
- 声音描述：水晶充能完毕的"叮"声

## 九、背景音乐（BGM）设计

### 9.1 音乐曲目列表

| 曲目 | 用途 | 风格 | 时长 | Intensity层叠 |
|------|------|------|------|--------------|
| BGM01 - 天使的摇篮曲 | 主菜单 | 管风琴 + 摇篮曲旋律 | 2:30循环 | 无 |
| BGM02 - 战斗序曲 | 战斗初期（1-10波） | 轻快弦乐 + 打击乐 | 1:45循环 | Intensity 1-3层 |
| BGM03 - 战斗高潮 | 战斗中期（11-20波） | 管弦乐全奏 + 合唱 | 2:00循环 | Intensity 4-6层 |

| BGM05 - 失败与希望 | 失败界面 | 钢琴 + 摇篮曲变奏 | 1:00一次 | 无 |

### 9.2 Intensity层叠系统

音乐根据**敌人数量**和**战斗激烈程度**自动层叠：

| Intensity层 | 触发条件 | 新增乐器 |
|------------|---------|---------|
| 层1 | 敌人<5 | 弦乐轻柔 |
| 层2 | 敌人5-10 | + 木管乐 |
| 层3 | 敌人10-20 | + 圆号 |
| 层4 | 敌人20-30 | + 小号 |
| 层5 | 敌人30-50 | + 定音鼓 |
| 层6 | 敌人50-80 | + 钹片 + 合唱 |
| 层7 | 敌人80-120 | + 管风琴 |
| 层8 | 敌人>120 | + 全管弦乐 + 全场合唱 |

**层叠过渡**：每次增减层用0.5秒淡入淡出，避免突兀。

### 9.3 Boss战音乐动态

Boss战音乐额外根据**Boss HP**动态变化：

| Boss HP范围 | 音乐表现 |
|------------|---------|
| 100%-70% | 正常Boss战音乐 |
| 70%-40% | 加入不安低音（大提琴拨弦） |
| 40%-10% | 音乐加速1.2倍 + 合唱增强 |
| <10% | 音乐达到最高潮（全管弦乐 + 合唱最强） |

## 十、音效实现技术规范

### 10.1 音效触发优先级

| 优先级 | 音效类型 | 行为 |
|--------|---------|------|
| 0（最高）| 婴儿恐惧音效 | 可打断其他音效 |
| 1 | Boss出场音效 | 可打断普通战斗音效 |
| 2 | 连携触发音效 | 不打断优先级0-1 |
| 3 | 卡牌释放音效 | 可同时播放多个 |
| 4（最低）| 环境音效 | 始终播放，可被遮盖 |

### 10.2 音效对象池

| 音效类型 | 对象池大小 | 说明 |
|---------|----------|------|
| 武器命中音效 | 10 | 高频触发，需要对象池 |
| 卡牌释放音效 | 5 | 中频触发 |
| 敌人受伤音效 | 8 | 高频触发 |
| UI音效 | 3 | 低频触发 |

### 10.3 音量规范

| 音效类型 | 默认音量 | 最大音量 | 备注 |
|---------|---------|---------|------|
| 婴儿情感音效 | 0.8 | 1.0 | 最重要，不可被遮盖 |
| 武器命中音效 | 0.6 | 0.8 | |
| 卡牌释放音效 | 0.5 | 0.7 | |
| 环境音效 | 0.3 | 0.5 | 背景层 |
| UI音效 | 0.4 | 0.6 | |
| BGM | 0.4 | 0.6 | 可被音效遮盖 |

## 十一、音效与玩法的关联设计

### 11.1 情感音效的玩法引导作用

| 情感状态 | 音效作用 | 玩家应做出的反应 |
|---------|---------|----------------|
| 好奇 | 轻松音效 → 玩家知道婴儿安全 | 可专注攻击/探索 |
| 恐惧 | 紧张音效 → 玩家立刻警觉 | 停止攻击，寻找婴儿 |
| 愤怒 | 爽快音效 → 玩家感到愉悦 | 可继续攻击，婴儿在帮忙 |
| 疲惫 | 柔和音效 → 玩家感到需要保护 | 靠近婴儿，使用恢复卡牌 |
| 觉醒 | 华丽音效 → 玩家感到兴奋 | 尽情输出，享受爽感 |

### 11.2 音效作为难度提示

| 情境 | 音效提示 | 玩家理解 |
|------|---------|---------|
| 敌人数量增多 | BGM Intensity层增加 | "战况激烈了" |
| Boss即将出现 | BGM逐渐过渡到Boss战 | "准备Boss战" |
| 婴儿精神力低 | 婴儿持续哭泣声增强 | "需要保护婴儿" |
| 连携即将触发 | 武器充能"嗡"声逐渐增强 | "准备触发连携" |
